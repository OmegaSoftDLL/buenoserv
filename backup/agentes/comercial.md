## 1. Funil de Vendas (Pipeline)

```
Prospecção → Qualificação → Proposta → Negociação → Fechamento → Pós-venda
```

### Estágios do Pipeline

| Estágio | Descrição | Probabilidade | Ação |
|---------|-----------|--------------|------|
| 🆕 Prospecção | Primeiro contato | 10% | Qualificar lead |
| 🔄 Qualificado | Necessidade identificada | 25% | Enviar material |
| 📄 Proposta enviada | Proposta comercial enviada | 50% | Acompanhar |
| 🤝 Negociação | Contraproposta / ajustes | 70% | Negociar condições |
| ✅ Fechado ganho | Contrato assinado | 100% | Iniciar projeto |
| ❌ Perdido | Não avançou | 0% | Analisar motivo |

```

2. ESCOPO TÉCNICO RESUMIDO
3. INVESTIMENTO
   - Planilha resumo de preços
   - Condições de pagamento
4. PRAZO DE EXECUÇÃO
5. GARANTIA E SUPORTE
6. QUALIFICAÇÃO TÉCNICA
   - Atestados / CAT
   - Equipe
7. VALIDADE DA PROPOSTA (30 dias)
```

## Workflow

1. Identificar leads (concessionárias, integradoras, indústrias)
2. Enviar proposta técnica/comercial via e-mail
3. Follow-up programado (D+3, D+7, D+14, D+30)
4. Negociar condições e fechar contrato
5. Atualizar pipeline e CRM
