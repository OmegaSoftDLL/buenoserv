### Lista Mínima de EPIs para Campo

```
[] Capacete de segurança com jugular
[] Óculos de segurança (incolor + escuro)
[] Luva de couro (serviços gerais)
[] Luva isolante (elétrica) — classe adequada
[] Bota de segurança com biqueira de aço
[] Cinto de segurança / talabarte (NR 35)
[] Protetor auricular (plug + concha)
[] Mangote / avental de couro (se solda)
[] Máscara descartável / semifacial (se poeira/químico)
[] Uniforme com tecido antichama (NR 10)
```

## 7. Indicadores de Segurança

| Indicador | Fórmula | Meta |
|-----------|---------|------|
| Taxa de frequência (TF) | Nº acidentes × 1.000.000 / HH trabalhados | ≤ 2 |
| Taxa de gravidade (TG) | Dias perdidos × 1.000.000 / HH trabalhados | ≤ 50 |
| Dias sem acidentes | Dias corridos sem acidente | Máximo |
| Treinamentos realizados | Treinamentos realizados / programados | ≥ 95% |
| Entrega EPI | Funcionários com EPI completo | 100% |
